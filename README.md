# Claude Skills: Knowledge

A collection of 15 Claude Code skills for Obsidian, daily journaling, task management, and knowledge systems.

## Skills

| Skill | Description |
|-------|-------------|
| `article-linker` | Map article titles to published URLs |
| `bug-record` | Record bug fixes to bugs.jsonl |
| `changelog` | Auto-generate daily changelogs (post-commit hook) |
| `daily-journal` | Daily journal generation and task recording |
| `diagram-indexer` | Incremental indexing for Mermaid/Canvas diagrams |
| `json-canvas` | Create and edit JSON Canvas (.canvas) files |
| `obsidian-bases` | Obsidian Bases database view operations |
| `obsidian-markdown` | Obsidian-flavored Markdown (wikilinks, callouts, properties) |
| `plan-writing` | Plan document formatting standards |
| `planning-with-files` | Manus-style persistent Markdown planning and tracking |
| `rss-daily-digest` | RSS feed daily digest and summary |
| `sync-notebooklm-kb` | Sync local articles to NotebookLM knowledge base |
| `task-checkpoint` | Auto-update plan + TODO + commit on task completion |
| `task-dashboard` | Scan active projects, extract and rank tasks by urgency |
| `task-sync-obsidian` | Sync completed tasks to Obsidian project pages |

## Install

```bash
git clone https://github.com/rongarede/claude-skills-knowledge.git ~/.claude/skills-repos/knowledge
cd ~/.claude/skills-repos/knowledge && bash install.sh
```

## Uninstall

```bash
cd ~/.claude/skills-repos/knowledge && bash uninstall.sh
```
